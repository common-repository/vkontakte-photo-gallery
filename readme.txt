== Description ==

Plugin allow import photos from social network vkontakte.ru.
<br>
Плагин позволяет импортировать фото из социальной сети vkontakte.ru.


== Installation ==

1. Upload plugin files to the `/wp-content/plugins/` directory.<br>
2. Activate the plugin through the 'Plugins' menu in WordPress.<br>
<br>
1. Загрузите файлы плагина в директорию `/wp-content/plugins/`.<br>
2. Активируйте плагин в панели администратора.<br>

== Frequently Asked Questions ==

= Как настроить плагин? =
Настраиват плагин не нужно. Он сам проверит наличие авторизации ВКонтакте, и при необходимости выдаст форму для ввода данных. Приложению необходим доступ к вашим фотографиям.
= Где взять русскую локализацию? =
Русскую локализацию можно скачать на <a href="http://bogutsky.ru/?page_id=216">http://bogutsky.ru/?page_id=216</a>.
= Почему я авторизован ВКонтакте под одним аккаунтом, а плагин дает выбрать фото из альбомов другого аккаунта? =
Вероятно Вы начали работать с плагином под другим аккаунтом. Для каждого сайта плагин запоминает аккаунт ВКонтакте. Чтобы изменить аккаунт нужно кликнуть на кнопку "Сменить пользователя" ("Change user").
